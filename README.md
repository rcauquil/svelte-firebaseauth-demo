# Svelte Firebase auth Template

## Config

### Firebase

You have to add your firebase config file under `config/default.json`.

```json
{
  "firebase": {
    "apiKey": "",
    "authDomain": "",
    "databaseURL": "",
    "projectId": "",
    "storageBucket": "",
    "messagingSenderId": "",
    "appId": "",
    "measurementId": ""
  }
}
```
